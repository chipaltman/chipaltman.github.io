Use pacman dash capital s to install a package by name.

To remove a package by name, use pacman dash capital r, s.

Use pacman dash capital s, s, to search for a package.

Pacman dash capital s, y, u, to upgrade all packages.

Use ﻿pacman dash capital q, i, to display local package information: Name, version, description, etc.

Pacman dash capital s, i, will display remote package information.

To list all packages with available upgrades, use pacman dash capital q, u.

The pacman command with no operator will show a list of any packages that are newly available in one of the repositories.

To list all dependencies of a package, use pacman dash bracket, capital s, pipe, capital q, bracket, i.

﻿

Use pacman dash capital d, k, to verify the dependencies of all installed packages. This is useful if an installation was killed.
